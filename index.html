    Welcome to my Github Pages

    My current project: bob-angular, an angular-three.js seed.

    The sample application uses angular, with less, bootstrap, jquery, three, d3, leaflet, and other libraries to enable the mapping of 2d Web to 3d surfaces.

    For a long time, I have been wanting to see a simple way of using a little 3D as needed. Over the last few years, as more and more hardware accelerated devices have moved to support HTML5 standards, CSS3 and JavaScript, what used to once take rendering farms of dedicated machines occurs in realtime on phones and tablets.

    Using three.js, and the CSS3DRenderer, both by Ricardo Cabello, distinct content types--video, map tiles, svg and html--are renderable in 3D space. Although this is merely the tip of the iceberg where 3D and 2D can meet on the Web, typical use case scenarios for 3D in Web content are for simple plane constructions.

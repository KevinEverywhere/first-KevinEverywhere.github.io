    Welcome to my Github Pages

    My current project, bob-angular, an angular-three.js seed, can be seen in a sample application here.

    The github files include a package.json, bower.json and testing packages. The application itself uses angular, with less, bootstrap, jquery, three, d3, leaflet, and other libraries to enable the mapping of 2d Web to 3d surfaces.

    As more and more hardware accelerated devices have moved to support HTML5 standards, CSS3 and JavaScript, what used to once take rendering farms of dedicated CPUs occurs in realtime on phones and tablets. Current HTML content is able to be rendered at runtime in 3D environment with open source software.

    Using three.js, and the CSS3DRenderer, both by Ricardo Cabello, distinct content types--video, map tiles, svg and html--are renderable in 3D space. Although this is merely the tip of the iceberg where 3D and 2D can meet on the Web, many of the most typical use case scenarios for 3D in Web content are simple plane constructions.
